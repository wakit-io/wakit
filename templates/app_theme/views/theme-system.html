Icon Button (아이콘 버튼)

아이콘만 표시하는 버튼입니다. 접근성을 위해 aria-label을 지정하세요.

버튼 예시 코드

<!-- Primary L -->
  <button type="button" class="btn btn--primary btn--l">확인</button>
  
  <!-- Secondary M + 아이콘 -->
  <a href="#" class="btn btn--secondary btn--m">
    <span>다음</span>
    <i class="bi bi-chevron-right"></i>
  </a>
  
  <!-- Tertiary S, 비활성 -->
  <button type="button" class="btn btn--tertiary btn--s" disabled>취소</button>
  
  <!-- Ghost -->
  <button type="button" class="btn btn--ghost btn--m">더보기</button>
  
  <!-- Block (전체 너비) -->
  <button type="button" class="btn btn--primary btn--l btn--block">제출</button>
  
  <!-- Icon Button (테마 컬러, L/M/S) -->
  <button type="button" class="btn btn--primary btn--l btn--icon" aria-label="찜"><i class="bi bi-heart"></i></button>
  <button type="button" class="btn btn--secondary btn--m btn--icon" aria-label="공유"><i class="bi bi-share"></i></button>
  <button type="button" class="btn btn--ghost btn--s btn--icon" aria-label="더보기"><i class="bi bi-three-dots"></i></button>

테마 컬러 아이콘 (icon-theme)

테마 색상을 적용한 아이콘입니다. 기본 40px, --sm 32px, --lg 48px 크기를 사용할 수 있습니다.

<!-- 기본(40px) -->
<div class="icon-theme"><i class="bi bi-graph-up-arrow"></i></div>
<!-- S(32px) / L(48px) -->
<div class="icon-theme icon-theme--sm"><i class="bi bi-heart"></i></div>
<div class="icon-theme icon-theme--lg"><i class="bi bi-star-fill"></i></div>
